# Домашняя работа 1

## word.py

Скрипт находит середину слова. Берём слово, смотрим сколько в нём букв. Если нечётное число — выдаём одну букву ровно посередине. Если чётное — две центральные.
Вычисляем индекс середины через целочисленное деление длины на 2. Для чётной длины вырезаем срез из двух символов (от mid-1 до mid+1), для нечётной - берём символ по индексу mid.


## tinder.py

Простенький tinder. Есть два списка - парни и девушки. Оба сортируются по алфавиту, а потом склеиваются через zip: первый с первой, второй со второй и так далее.

Если списки разной длины , то программа ругнётся, что кто-то останется без пары.


## Быстрый старт 

```bash
python3 word.py
python3 tinder.py
```
