# Домашки по Python (ВШЭ)

Тут лежат мои решения домашних заданий по курсу Python. Ничего сверхъестественного, просто базовые задачки для понимания основ.

## Что тут есть

### HM1 - Первые шаги

**word.py** - находит середину слова. Если длина чётная, выводит два символа из середины, если нечётная - один. Простенько, но пришлось подумать над индексами.

**tinder.py** - тиндер для списков имён. Сортирует мальчиков и девочек по алфавиту и составляет пары. Если кого-то не хватает - ругается.

### HM2 - Функции

**task1.py** - функция `sum_distance`, которая считает сумму всех чисел между двумя значениями. Работает в обе стороны, так что порядок аргументов неважен.

**task2.py** - `trim_and_repeat` - обрезает строку слева и повторяет её нужное количество раз. Удобно для всяких манипуляций с текстом.

### HM3 - Работа с файлами

**task1.py** - парсит лог покупок из txt-файла и собирает всё в словарь. Ключ - id юзера, значение - категория товара.

**task2.py** - строит воронку продаж. Берёт данные о визитах и покупках, объединяет их и сохраняет в новый csv. По сути, находит пересечение между теми, кто зашёл и теми, кто купил.

### HM4 - Даты

**hm4_dates.ipynb** - парсинг дат из разных газет. У каждой газеты свой формат записи даты, и нужно было разобраться с `datetime.strptime()` и всеми этими `%A`, `%B`, `%Y`. Газеты: The Moscow Times, The Guardian, Daily News.
